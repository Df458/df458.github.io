DFGame

DFGame is a lightweight game development framework written in C and Vala. Unlike a full-blown game engine, DFGame uses a mostly flat code structure in order to allow game and tool developers to use the routines it provides in whatever configuration they want. The framework's primary goal is to provide a set of simple libraries that allow developers to rapidly produce small, tightly scoped engines and tools for their game projects.
