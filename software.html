Halberd

Halberd is a work-in-progress RPG engine. The goal of the project is to create an accessible and free tool to help designers build old-school JRPGs. In addition to this, Halberd is also being built to support many modern features and effects.

Halberd is being built on top of the DFGame framework, and uses GTK+ for its interface.

Sorthelper

Sorthelper is a file-sorting utility built with GTK+. It uses efficient as-you-type searching combined with file previews and grouping algorithms to help users rapidly sort their files.

Singularity

Singularity is a GTK3 webfeed reader for the desktop. It has support for RSS, RDF, and Atom feeds. It can also import/export OPML files to use in other feed readers.

The program does not rely on a server or any web services, instead handling everything client-side.

DFGame

DFGame is a simple game development framework written in C and Vala. It values simplicity and flexibility over ease of use, giving dedicated programmers a head start when building their game's engine code.

DFLib

DFLib is a small Vala-based utility library for GTK+ programs. It provides a few useful widgets and patterns to help speed up GUI development.
