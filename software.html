Singularity

Singularity is a GTK-based webfeed aggregator for the desktop. It is written in Vala. Currently, only linux is supported, although I may consider porting the code to target other platforms in the future.

Sorthelper

Sorthelper is a GTK-based file-sorting utility. It uses efficient as-you-type searching combined with file previews and grouping algorithms to help users rapidly sort their files.
